# starter-npm-cli
========================

A project generator to develop a npm CLI package.

## Installation

npm install -g starter-npm-cli

## Usage

```
starter-npm-cli [Author_Name] [Package_Name]
```

## Directory structure

```
[Package_Name]
├── LICENSE        // Default license is MIT.
├── README
├── bin 
│   └── index.js   // CLI module.
└── package.json
```
